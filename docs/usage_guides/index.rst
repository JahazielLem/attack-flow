Usage Guides
============

The Attack Flow usage guides outline how to build clear, well-structured attack
flows. It provides guidance on flow structure, documentation, and quality
standards, along with real-world uses in cyber threat intelligence, defensive
posture analysis, and red teaming. The goal is to help teams visualize adversary
behavior, identify detection gaps, and improve communication across
threat-informed operations.
